# 02.算數運算符
| 運算符 | 描述 | 實例|
| ---- | ---- | ----|
| + | 加 | 10 + 20 = 30|
| - | 減 | 10 - 20 = -10|
| * | 乘 | 10 * 20 = 200|
| / | 除 | 10 / 20 = 0.5|
| ** | 次方 | 2 ** 3 = 8|
| // | 取整數 | 10 // 3 = 3|
| % | 取餘數 | 10 % 3 = 1|

## 範例 
### Code
```python
print (10 + 20)
print (10 - 20)
print (10 * 20)
print (10 / 20)
print (2 ** 3)
print (10 // 3)
print (10 % 3)
```
### OUTPUT
```
30
-10
200
0.5
8
3
1
```
