# 07.變數類型轉換
| 類型 | 描述 | 實例 |
| ---- | ---- | ----|
| int( ) | 將x轉換為整數(無條件捨去) | int(x)|
| float( ) | 將x轉換為浮點數 | float(x)|
| str( ) | 將x轉換為字串 | str(x)|
| |還有一些 我有點懶 | |

### Code
```python
x = 10.7    
print (int(x))      #將x轉換為整數(無條件捨去)
print (float(x))    #將x轉換為浮點數
print (str(x))      #將x轉換為字串
```
### OUTPUT
```
10
10.7
10.7
```
