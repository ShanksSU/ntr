# 04_0變量類型
| 類型 | 描述 | 實例 |
| ---- | ---- | ----|
| int | 整數 | test = 10|
| float | 浮點數 | test = 10.5|
| str | 字符串 | test = "panda"|
| bool | 布林 | test = True|

## 範例 
### Code
```python
name = "ayaka"              #字符串
age = 18                    #整數
gender = "girl"             #字符串
height = 157.84             #浮點數
```
