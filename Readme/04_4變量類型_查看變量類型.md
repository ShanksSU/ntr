# 04_4.變量類型_查看變量類型

| 類型 | 描述 | 實例 |
| ---- | ---- | ----|
| type( ) | 查看x的變量類型 | type(x)|

### Code
```python
x = 10    
x01=10.5
x02="Chi"

print (type(x))     #利用type()來查看x的變量類型
print (type(x01))   #利用type()來查看x01的變量類型
print (type(x02))   #利用type()來查看x02的變量類型
```
### OUTPUT
```python
<class 'int'>
<class 'float'>
<class 'str'>
```
